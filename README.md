# Books
Books - это API приложение, которое позволяет просматривать список книг, доступных на сайте, просматривать детали определенной книги и много другое.


## Установка
1. Клонируйте репазиторий с GitHub
2. Создайте виртуальное окружение
3. Установите зависимости "pip install -r requirements"
4. Создайте таблицы в базе данных
5. Выполните все нужные миграции
6. Запустите проет "python manage.py runserver"
